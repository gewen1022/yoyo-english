# 佑佑的英语小世界

记录佑佑每天学英语的进度，支持离线使用。

## 功能

- **每日打卡** — 英文儿歌、日常蹦词、绘本共读、学习APP、互动游戏
- **反应记录** — 开心配合 / 一般般 / 不太想参与
- **新词记录** — 记录每天新学的单词，自动统计累计单词数
- **连续打卡** — 自动计算连续天数
- **月历视图** — 一眼看到本月哪些天打了卡
- **历史记录** — 查看最近打卡详情，支持单条删除
- **补录功能** — 忘了打卡可以补
- **数据备份** — 导出/导入 JSON，换手机不丢数据

## 使用方式

手机 Chrome 浏览器打开 → 菜单 → "安装应用" → 桌面上就有图标了。

## 技术

纯前端 PWA，数据存手机本地，无需服务器。
